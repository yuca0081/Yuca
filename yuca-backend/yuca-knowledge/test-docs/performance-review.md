# 绩效考核管理办法

> 本办法规范星河科技有限公司员工绩效考核的周期、流程、结果应用，旨在通过公平、透明、可量化的评估机制，激励员工持续成长并支撑公司战略目标落地。自 2025 年度起执行。

## 一、考核原则

### 1.1 目标导向

考核以目标达成度为核心，结合过程表现与价值观践行，避免单一以工作量为标准的偏向。

### 1.2 公平公正

- 考核标准公开透明，员工入职即知晓
- 评估基于事实与数据，避免主观臆断
- 员工有权知晓评估依据并提出申诉

### 1.3 差异化激励

考核结果与薪酬、晋升、培训机会强挂钩。优秀的加倍激励，不合格的及时改进或退出。

## 二、考核周期与范围

### 2.1 考核周期

| 考核类型 | 周期 | 适用范围 |
|---|---|---|
| 月度考核 | 每月 | 全体员工 |
| 季度 OKR 复盘 | 每季 | 全体员工 |
| 半年度考核 | 上半年 / 下半年 | 全体员工 |
| 年度考核 | 全年 | 全体员工 |
| 试用期考核 | 转正前 | 试用期员工 |

### 2.2 适用人员

所有正式员工、试用期员工、外包人员均参与考核。实习生参与月度反馈但不强制评级。

## 三、考核内容

### 3.1 业绩目标（OKR）— 占 70%

员工每季度初与直属上级共同制定 OKR（Objectives and Key Results）：
- **O（Objective）**：方向性、有挑战性的目标，建议 2-3 个
- **KR（Key Result）**：可量化的关键结果，每个 O 下 2-4 个

季度末由员工自评、上级评估，最终得分按达成度计算：
- 0.7 分为合格（达到预期）
- 0.8-1.0 分为优秀
- < 0.5 分为不合格

### 3.2 价值观与行为 — 占 30%

公司价值观 6 条：
1. **客户第一**：决策时优先考虑客户价值
2. **拥抱变化**：主动适应业务调整和组织变革
3. **团队合作**：跨部门协作顺畅，乐于分享
4. **诚信**：言行一致，数据真实
5. **敬业**：主动承担，结果导向
6. **持续学习**：知识更新，技能提升

每条价值观按"超越 / 符合 / 待改进"三档评估，由上级评分 + 同事反馈（360 度）综合得出。

### 3.3 加分项

以下情形可在最终得分基础上加分（最高加 5 分）：
- 主导重大项目取得突破性成果
- 获得客户书面表扬
- 在技术 / 管理上有创新并落地
- 培养新人成效显著
- 在突发事件中表现突出

## 四、考核等级

### 4.1 强制分布

半年度 / 年度考核按以下等级强制分布：

| 等级 | 描述 | 比例 |
|---|---|---|
| S | 卓越 | ≤ 5% |
| A | 优秀 | 20-25% |
| B+ | 良好 | 30-35% |
| B | 符合预期 | 30-40% |
| C | 待改进 | ≤ 5% |
| D | 不胜任 | ≤ 3% |

部门人数不足 10 人的，S 和 D 名额可空缺。

### 4.2 等级与得分对应

| 综合得分 | 等级 |
|---|---|
| ≥ 95 | S |
| 90-94 | A |
| 80-89 | B+ |
| 70-79 | B |
| 60-69 | C |
| < 60 | D |

## 五、考核流程

### 5.1 目标设定（季初）

1. 员工草拟 OKR，提交至上级
2. 1-on-1 沟通，调整并确认
3. OKR 录入绩效系统，对部门内公开

### 5.2 过程跟进（季度中）

- 周例会汇报 KR 进展
- 月度 1-on-1 反馈
- 季中复盘，必要时调整 KR（O 不可改）

### 5.3 季末评估（季末）

1. **员工自评**（2 个工作日）：填写完成度、亮点、反思
2. **上级评估**（2 个工作日）：基于事实数据给出评分与评语
3. **同事反馈**（同步）：360 度匿名反馈
4. **校准会议**（部门内）：部门负责人主持，确保跨团队公平
5. **结果沟通**：上级与员工 1-on-1，员工确认或申诉

### 5.4 申诉机制

员工对考核结果有异议的，可在结果公布后 5 个工作日内：
1. 先与上级沟通
2. 沟通无果的可向部门 HRBP 申诉
3. 仍不满意的向 HR 总监申诉，HR 总监裁决为最终结果

## 六、结果应用

### 6.1 薪酬调整

年度考核结果与年度调薪挂钩：

| 等级 | 调薪幅度 |
|---|---|
| S | 15-25% |
| A | 10-15% |
| B+ | 5-10% |
| B | 0-5% |
| C | 不调薪 |
| D | 不调薪，进入改进计划 |

### 6.2 年终奖

年终奖 = 月薪 × 系数 × 在职月份 / 12

| 等级 | 系数 |
|---|---|
| S | 4-6 个月 |
| A | 3-4 个月 |
| B+ | 2-3 个月 |
| B | 1.5-2 个月 |
| C | 0.5-1 个月 |
| D | 0 |

### 6.3 晋升

- 半年度连续 2 次 A 及以上：可申请晋升答辩
- 年度 S：优先进入晋升通道
- 连续 2 次 D：不胜任退出

### 6.4 不胜任处理（PIP）

考核等级 C 或 D 的员工进入 Performance Improvement Plan（绩效改进计划）：
- 时长：1-3 个月
- 内容：明确改进目标、每周 1-on-1 辅导
- 结果：改进成功恢复正常状态；改进失败协商解除劳动合同（N+1 补偿）

## 七、特殊情形

### 7.1 调岗

考核周期内调岗的：
- 调岗前工作满 2 个月：原部门给出评级，按时间比例综合
- 调岗前工作不满 2 个月：由新部门主导评估

### 7.2 长期请假

- 当季度请假超过 20 个工作日的，本季度不评级，年度按实际在职季度综合
- 产假期间不参与评级，薪资按国家规定发放

### 7.3 新入职

- 入职满 3 个月参与当季评估但不强制评级
- 入职未满 3 个月不参与年度评级，年终奖按月折算

## 八、附则

1. 本办法由人力资源部负责解释
2. 考核数据属 L3 机密，仅本人、直属上级、HR 可查阅
3. 本办法自 2025 年度起执行，原《绩效考核细则》同时废止
